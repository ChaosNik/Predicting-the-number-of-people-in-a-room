Machine Learning (ML). Describe classification and regression ML algorithms and their
typical representatives. Preparing datasets for training, validation, and testing.
In the practical part of the work, analyze models for predicting the number of people
in a room created using several algorithms (classification and regression).
The models need to be trained with data containing temperature, humidity, CO2 levels,
etc. Use the Jupyter Notebook tool for implementation. Perform a comparative analysis
of the performance of the obtained models.

How to open Jupyter Notebook:
	1. Create Python virtual env in ./Practical part by executing "python -m venv ./Practical part/.venv"
	2. Activate virtual env by executing "./Practical part/.venv/Scripts/activate"
	3. Install all requirements from ./Practical part/requirements.py by executing "pip install -r ./Practical part/requirements.txt"
	4. Open Jupyter notebook by executing "jupyter notebook"
